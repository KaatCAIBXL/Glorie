# Invitation mariage — Gloire & Annabelle

Site d'invitation de mariage (lettre d'ouverture, compte à rebours, sections plein écran).

## Ouvrir localement

Ouvrez `index.html` dans votre navigateur, ou servez le dossier avec un serveur local.

## Publier sur GitHub Pages

1. Poussez ce dépôt sur GitHub (voir ci-dessous).
2. Sur GitHub : **Settings → Pages → Build and deployment → Source** : choisissez la branche `main` et le dossier `/ (root)`.
3. Le site sera disponible à : `https://<votre-utilisateur>.github.io/<nom-du-repo>/`
